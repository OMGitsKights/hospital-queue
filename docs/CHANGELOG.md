# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - Initial Release

### Added
*   **Authentication**: Login and Registration for Patients, Doctors, and Admins.
*   **Queue System**:
    *   Book appointments for specific slots.
    *   View real-time queue status.
    *   Admin capability to reset queues.
*   **Doctor Portal**:
    *   View assigned patients.
    *   Mark appointments as "Done".
    *   Add digital prescriptions.
*   **Patient Portal**:
    *   "My Bookings" history.
    *   View prescriptions.
*   **Localization**: Support for English, Hindi, and Telugu.
*   **Documentation**: Initial documentation set (README, API, Architecture).
